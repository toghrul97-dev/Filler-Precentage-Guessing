# Fill Percentage Guessing Game

## Overview
This is a browser-based guessing game where the player tries to estimate what percentage of a container is filled with a colored liquid.  
The game randomly selects a fill percentage, and the player must guess how full the container is.

## How the Game Works
1. A container is displayed on the screen.
2. The game randomly fills the container to a certain percentage.
3. The player guesses the percentage of the container that is filled.
4. The game reveals the actual percentage.
5. The player can see how close their guess was and try again.
